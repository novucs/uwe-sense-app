# UWE Sense Phone App
Phone application for the UWE Sense project.

## Prerequisites
- NodeJS and NPM
- NativeScript
- (Optional - Required for Android) Android SDK
- (Optional - Required for iOS) iOS SDK and an Apple PC / MacBook

## Installation
### Android
#### Compile

`tns build android`

#### Run

`tns run android`

### iOS
#### Compile

`tns build ios`

#### Run

`tns run ios`
